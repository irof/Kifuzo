# Kifuzo (棋譜蔵)

Kifuzoは、macOS向けの将棋棋譜管理・閲覧アプリケーションです。
Compose Multiplatform (Kotlin) を使用して構築されており、ローカルの棋譜ファイルの整理や閲覧を快適に行うことができます。

## 主な機能

*   **ファイルブラウザ**: ディレクトリ構造を維持したまま棋譜ファイルを素早く探索。
*   **盤面表示**: 最終局面の自動表示、反転表示、持駒の標準順序表示に対応。
*   **棋譜解析**: 評価値グラフの表示、急激な評価値変化（疑問手・好手）の自動強調。
*   **戦型検知**: 棋譜から「矢倉」「ゴキゲン中飛車」などの戦型を自動判別。
*   **インポート・変換**: 将棋クエスト等の棋譜インポート、CSA形式からKIF形式への変換。
*   **リネーム機能**: 対局日、対局者、戦型に基づいたファイルの一括リネーム。

## 動作環境

*   **OS**: macOS (Apple Silicon / Intel)
*   **JDK**: 25 以上

## 実行・ビルド方法

プロジェクトのルートディレクトリで以下のコマンドを実行してください。

### アプリケーションの起動
```bash
./gradlew run
```

### macOS向け配布パッケージ (.dmg) の作成
```bash
./gradlew packageDmg
```

### テストと検証の実行
```bash
./gradlew verify
```
