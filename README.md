# Kifuzo (棋譜蔵)

Kifuzoは、macOS向けの将棋棋譜管理・閲覧アプリケーションです。
Compose Multiplatform (Kotlin) を使用して構築されており、ローカルの棋譜ファイルの整理や閲覧を快適に行うことができます。

## 主な機能

*   **ファイルブラウザ**: ディレクトリ構造を維持したまま棋譜ファイルを素早く探索。
*   **盤面表示**: 最終局面の自動表示、反転表示、持駒の標準順序表示に対応。
*   **棋譜解析**: 評価値グラフ、消費時間グラフの表示、急激な評価値変化（疑問手・好手）の自動強調。
*   **貼り付け・インポート**: クリップボードからの棋譜貼り付け（自動形式判別）、将棋クエスト等の棋譜インポート。
*   **堅牢なパース**: 拡張子に関わらず内容からKIF/CSA形式を自動判別。形式不一致時の警告表示機能。
*   **リネーム・変換**: 対局日、対局者に基づいたファイルの一括リネーム。CSA形式からKIF形式への変換。

## 動作環境

*   **OS**: macOS (Apple Silicon / Intel)
*   **JDK**: 25 以上

## 実行・ビルド方法

プロジェクトのルートディレクトリで以下のコマンドを実行してください。

### アプリケーションの起動
```bash
./gradlew run
```

### 解析・レポートの実行
コードの品質（複雑度等）の測定やテストカバレッジの確認が可能です。

*   **静的解析 (detekt)**: `build/reports/detekt/detekt.html` にレポートが生成されます。
    ```bash
    ./gradlew detekt
    ```
*   **カバレッジ測定 (Kover)**: `build/reports/kover/html/index.html` にレポートが生成されます。
    ```bash
    ./gradlew koverHtmlReport
    ```
*   **一括検証 (Spotless + Test + detekt + Kover)**:
    ```bash
    ./gradlew verify
    ```
