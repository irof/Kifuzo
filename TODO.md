# Kifuzo Project TODO / Technical Debt

## 技術的負債 (Technical Debt)
- [ ] **インポートの完全な最適化**: ワイルドカードインポートは解消したが、`sed` による機械的置換のため、一部のファイルに不要なインポートが残っている可能性がある。`spotlessApply` で消しきれなかった重複や未使用のインポートを手動確認する。
- [ ] **CsaParser / KifuParser のクラス化**: 現在これらはトップレベル関数として実装されているが、状態管理や拡張性を考慮すると、インターフェースを定義したクラス構造にリファクタリングする余地がある。
